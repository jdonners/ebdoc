## Introduction
The Open MPI Project is an open source MPI-2 implementation. 

OpenMPI is available using the command:

```
module load OpenMPI
```

* [More about OpenMPI](http://www.open-mpi.org/)
* [More about modules](Local:/systems/lisa/software/modules)
